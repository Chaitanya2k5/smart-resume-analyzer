# Smart Resume Analyzer

An ATS-style Resume Analyzer built using Python and Streamlit.

## Features
- PDF Resume Upload
- ATS Compatibility Score
- Resume Match Score
- Role-Based Skill Analysis
- Missing Skill Detection
- Improvement Suggestions

## Technologies Used
- Python
- Streamlit
- PyPDF2

## Job Roles Supported
- Software Developer
- Data Analyst

## How to Run

```bash
python -m streamlit run app.py
```
